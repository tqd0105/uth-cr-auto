# UTH Portal Bridge

Chrome extension này đồng bộ dữ liệu từ tab `https://portal.ut.edu.vn` đã đăng nhập về app local.

## Cài đặt

1. Mở Chrome `chrome://extensions`.
2. Bật `Developer mode`.
3. Chọn `Load unpacked`.
4. Chọn thư mục `extension` trong repo này.

## Cách dùng

1. Đăng nhập app local ở `http://localhost:3000`.
2. Vào dashboard, bấm `Copy token`.
3. Mở `https://portal.ut.edu.vn` và đăng nhập portal.
4. Bấm extension `UTH Portal Bridge`.
5. Dán bridge token.
6. Bấm `Sync dashboard`.
7. Quay lại app và bấm refresh.

Khi cần xem lớp của một học phần, nhập mã học phần trong extension rồi bấm `Sync lớp`.
